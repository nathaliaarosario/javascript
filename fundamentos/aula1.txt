Particularidades da Linguagem (Sintaxe básica)
1) No JavaScript não usamos o conceito de linha e sim sentença

2) ; é opcional

3) Podemos usar "" ou '' (não pode misturar)
    Dica: usar "" para textos e '' para importações(de bibliotecas)

4) No JavaScript existe 3 tipos de variáveis (String, number e boolean)
    String (caracteres)
    Number (números inteiros e não inteiros)
    Boolean (verdadeiro ou falso)

5) No JavaScript as variáveis são dinâmicamente tipadas, ou seja o tipo é determinado quando o valor é atrinuído
